## üìå Project Overview

This dashboard was designed to analyze how industrial robotic systems behave during production, combining operational execution logs with mechanical telemetry such as speed, acceleration, and joint torque.

The solution is structured around a star schema data model, separating fact tables (task execution, movement telemetry, torque readings) from dimension tables (robot models, tasks, calendar), ensuring scalability, performance, and professional data modeling practices.
